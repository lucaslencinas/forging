# Esports Data Sources

## Primary Data Sources

### Prize Pool Data
- **Esports Earnings** - https://www.esportsearnings.com/
  - Historical prize money by game, player, tournament
  - Player counts and tournament counts
  - Best for: All-time and yearly prize pool data

- **Esports Charts** - https://escharts.com/
  - Prize pools + viewership data
  - Peak viewers per tournament
  - Best for: Viewership analytics

- **Newzoo** - https://newzoo.com/

  - Annual esports reports

### Other Useful Sources
- **Liquipedia** - https://liquipedia.net/
  - Wiki-style tournament/player data
  - Good for specific game ecosystems

- **Esports Insider** - https://esportsinsider.com/
  - Industry news and analysis
  - Marketing statistics
